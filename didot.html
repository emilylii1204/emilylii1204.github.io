Didot Typeface

WHAT IS IT?

A typography specimen book that not only highlights the beautiful details of the Didot font family, but also using the typeface to create artworks.

COLLECTIONS

The book is divided into categories: number, symbols, weight, size, mathematics, diacritics and ligature.

NEGATIVE SPACES

Adding on cut paper to illustrate highlights the shape of the typeface and allowing lighting to create its own artwork.

MATERIALS

Using transparent paper to see the small adjustments between weights.

ILLUSTRATIONS

Simple line drawings printed on the first and separated layer complimenting with the contrasting strokes of the typeface, at the same time does not take away the subject: the typeface.

THE ARTWORKS

Using the tyepface to create non-objective artwork through reptition, simple transformation, different sizes.

LAYOUT OVERVIEW

each collection reflects the function of the symbol, number and letters.
